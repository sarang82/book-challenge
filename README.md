# 📚 톡독(TalkDok) – 독서 챌린지 앱
**개발 기간:** 2025.03 ~ 2025.06  
1020 세대의 독서를 더 재미있고 동기부여 있게 만들어주는 독서 챌린지 어플리케이션입니다.

---

## 🔎 프로젝트 기획 배경

2025년 3월 디지털데일리 기사에 따르면 YES24 조사에서 **1020 세대의 도서 구매량이 전년 대비 18% 증가**했습니다.  
독서는 단순 지식 습득을 넘어 **문화 소비**이자 **자기표현의 수단**이 되면서, 경험과 공유를 중시하는 Z세대의 독서 문화를 지원할 수 있는 서비스의 필요성이 커졌습니다.

또한 **통계청 자료 기준 20대 이하의 독서 목적**은 다음과 같습니다.
- 재미를 위해
- 마음의 성장을 위해
- 자기 개발을 위해

> 이러한 요구를 반영하여 **‘흥미 · 동기 · 성장’**을 중심으로 한 독서 챌린지 앱 **톡독(TalkDok)** 을 기획했습니다.

---

## 🎯 주요 기능

### ✔ 독서 챌린지
- 한 권의 책을 완독하는 것을 목표로 한 챌린지 제공

### ✔ 독서 미션
- 매일 1개의 **랜덤 독서 미션** 제공  
- 미션으로 꾸준한 독서 습관 형성

### ✔ 내 서재
- 읽는 책 / 읽은 책 / 관심 도서 관리 기능

### ✔ 도서 리뷰
- 개별 도서 리뷰 작성 및 조회

### ✔ 동기 부여 메시지
- 하루에 한 번 제공되는 응원 및 명언 메시지

### ✔ 독서 타이머
- 집중 독서를 위한 타이머 기능 및 기록 저장

### ✔ AI 도서 추천
- 사용자의 취향 기반 맞춤형 도서 추천
